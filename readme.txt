== Description ==

Amazing Team Member Carousel is a super modern team member plugin. Unlimited Colors, 100% responsive, automatic resize images, flexible, unlimited items, shortcode powered, custom link and more

You can easily add team members & embed them in separate page post or widgets.

== Installation ==

Installing this plugin as regular wordpress plugin.

After install, this plugin enable a custom post called <strong>Team Members</strong>. Add new items on that custom post. 
After adding items, you are ready to use this plugin in your post, page, widget or anywhere you like. 

Go to page > Add New. On the editor, you will see a button called <strong>Team Shortcode</strong> by clicking button you see the shortcode like <code>[atmc]</code>
